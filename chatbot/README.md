# Rule-Based Chatbot

This project is a simple rule-based chatbot developed using Python.

## Description
The chatbot responds to user input using predefined rules and if-else conditions.
It can answer basic questions and simulate a simple conversation.

## Technologies Used
- Python

## How It Works
- The user enters a message
- The chatbot matches the input with predefined rules
- Based on the rule, an appropriate response is displayed

## Learning Outcome
- Understanding basic Natural Language Processing
- Learning conversation flow using conditional logic
