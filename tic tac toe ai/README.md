# Tic-Tac-Toe AI

This project is a Tic-Tac-Toe game where a human player plays against an AI.

## Description
The AI player uses the Minimax algorithm to make optimal decisions.
This makes the AI unbeatable.

## Technologies Used
- Python

## How It Works
- The user plays as X
- The AI plays as O
- The AI evaluates all possible moves using Minimax
- The best move is selected automatically

## Learning Outcome
- Understanding game theory
- Learning search algorithms
- Implementing decision-making logic
